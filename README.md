# my_configs

## Guide

### Install repo
```bash
git clone https://github.com/Kucukcollu/my_configs.git
```

### Install deps
```bash
cd my_configs
bash install_ws.sh
```

### set zsh config
```bash
cat .zshrc > ~/.zshrc
```

### set tmux config
```bash
cat .tmux.conf > ~/.tmux.conf
```

### set vim config
```bash
cat .vimrc > ~/.vimrc
```

### Additional ubuntu settings
```bash
sudo visudo

# %sudo   ALL=(ALL:ALL) NOPASSWD:ALL
```
